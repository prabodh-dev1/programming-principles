## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or pnpm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/prabodh-dev1/programming-principles.git
cd programming-principles
```

2. Install dependencies:
